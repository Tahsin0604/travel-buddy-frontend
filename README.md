## Getting Started

First install dependencies

````bash
npm install

than run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
````

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# _Featured_

---

- User have to login to add new post or to send request to join a trip
- Root admin can create a account for another admin and and monitor all the users and post
- Only user can create an post and send request
- for a post user can upload maximum 5 photos
- User can mange its post and trip request from his dashboard
